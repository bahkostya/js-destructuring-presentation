Destructuring in JavaScript

Destructuring in

What is it?

The destructuring assignment syntax is a JavaScript expression that makes it possible to unpack values from arrays, or properties from objects, into distinct variables.

  • Array destructuring
  • Object destructuring

Array destructuring

Basic variable assignment

			const iterable = ['a', 'b'];
			const [x, y] = iterable;
			console.log(x); // 'a'
			console.log(y); // 'b'

Array destructuring

Assignment separate from declaration

			const [a, b] = [1, 2];
			console.log(a); // 1
			console.log(b); // 2

Array destructuring

Default values

			[a=5, b=7, c] = [1];
			console.log(a); // 1
			console.log(b); // 7
			console.log(c); // undefined

Array destructuring

Swapping variables (without destructuring)

			let one = 'first';
			let two = 'second';
			
				let temp = one;
				one = two;
				two = temp;
				console.log(one); // 'second'
				console.log(two); // 'first'

Array destructuring

Swapping variables (with destructuring)

			let one = 'first';
			let two = 'second';
			
				[two, one] = [one, two];
				console.log(one); // 'second'
				console.log(two); // 'first'

Array destructuring

Parsing an array returned from a function

			const f = () => [1,2];
			const [a, b] = f();
			
				console.log(a); // 1
				console.log(b); // 2

Array destructuring

Ignoring some returned values

			const f = () => [1,2,3,4,5];
			const [a, , , , b] = f();
			
				console.log(a); // 1
				console.log(b); // 5

Array destructuring

Assigning the rest of an array to a variable

			const [a, b, ...rest] = [10, 20, 30, 40, 50];
			console.log(a); // 10
			console.log(b); // 20
			console.log(rest); // [30, 40, 50]

Object destructuring

Basic assignment

			const obj = { p: 42, q: true };
			const { p, q } = obj;
			console.log(p); // 42
			console.log(q); // true

Object destructuring

Assignment without declaration

			const { a, b } = { a: 1, b: 2 };
			console.log(a); // 1
			console.log(b); // 2

Object destructuring

Assigning to new variable names

			const { a: foo, b: bar } = { a: 1, b: 2 };
			console.log(a); Uncaught ReferenceError: a is not defined
			console.log(b); Uncaught ReferenceError: b is not defined
			console.log(foo); // 1
			console.log(bar); // 2

Object destructuring

Default values

			const { a = 10, b = 5 } = { a: 3 };
			console.log(a); // 3
			console.log(b); // 5

Object destructuring

Assigning to new variables names and providing default values

			const { a:aa = 10, b:bb = 5 } = { a: 3 };
			console.log(aa); // 3
			console.log(bb); // 5

Nested object and array destructuring

			const metadata = {
			  title: 'a pen'
			  translations: [
				  {
				    locale: 'de',
				    title: 'kugelschreiber',
				  }
			  ]
			}
			const {title: englishTitle, translations: [{title: germanTitle}]} = metadata;
			console.log(englishTitle); // 'a pen'
			console.log(germanTitle); // 'kugelschreiber'

Computed object property names and destructuring

			let key = 'z';
			let {[key]: foo} = {z: 'bar'};
			console.log(foo); // 'bar'

Rest in Object Destructuring

			let {a, b, ...rest} = {a: 10, b: 20, c: 30, d: 40}
			console.log(a); // 10
			console.log(b); // 20
			console.log(rest); // let {a, b, ...rest} = {a: 10, b: 20, c: 30, d: 40}
